## Labs & Capstones

- `CAPSTONE` Supervised Research Agent - Planner -> Researcher -> Summariser -> Reviewer with source logging and human approval
- Lab: Build one agent skill for code review, data cleaning, or report generation
- Lab: Add one hook - block unsafe commands, run formatter/tests, or validate generated JSON
- Lab: Chrome CDP/Playwright capture - network logs, screenshot, console errors, extracted data
- Lab: Coding agent PR - give an agent a small issue, require tests, review the diff
