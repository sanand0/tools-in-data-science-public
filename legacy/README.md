# Legacy Course Pages

This folder contains older top-level lesson pages that were historically published from the repository root.

They are kept here as source files so the root of the repository can stay focused on the current term and publishing configuration. The Hugo build republishes these pages at their original root URLs and under previous term URLs, preserving existing links from old sidebars, bookmarks, and discussions.

New course content should go under the active term folder, for example `2026-05/docs/`.
